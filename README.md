# Node-ecom

An eccomerce website with a landing page of products and cart . And administrattor (signin, signup) page  with production level authentication . 

To use clone the repo, 
1. run :   npm install
2. run :   npm run dev


Languages used:
HTML, CSS, JAVASCRIPT

External libraries used:
font-awesome, bulma
